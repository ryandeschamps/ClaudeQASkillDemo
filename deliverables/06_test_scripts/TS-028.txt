TEST SCRIPT: TS-028
Test Scenario: Enter Shipping Address
User Story: As a Buyer in Checkout, I want to provide my shipping address, so that I can receive my order at the desired location
Priority: Critical
Related Requirements: FR-008

GIVEN the user is logged in
  AND the user has completed billing address step
  AND the user is on the checkout page - shipping address step

WHEN the user enters shipping address information:
  - Full name
  - Address line 1
  - Address line 2 (optional)
  - City
  - State/Province
  - ZIP/Postal code
  - Country
  - Phone number
  AND submits the form

THEN the system should:
  - Validate all required fields are completed
  - Validate shipping availability to the entered PIN code
  - Validate format of phone number and postal code
  - Save the shipping address
  - Calculate shipping charges based on address
  - Proceed to next checkout step (order review)

WHEN the user selects a saved shipping address from address book

THEN the system should populate all fields automatically
  AND validate shipping availability
  AND calculate shipping charges
  AND allow the user to proceed

WHEN shipping is not available to the entered PIN code

THEN the system should display an error message
  AND prevent proceeding until a valid address is entered

EXPECTED RESULT:
- All shipping address fields are captured correctly
- Shipping availability is validated
- Shipping charges are calculated accurately
- Saved addresses can be selected and used
- Option to save new address for future use is available
- PIN code validation prevents undeliverable addresses
