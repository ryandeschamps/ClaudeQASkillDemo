TEST SCRIPT: TS-025
Test Scenario: Remove Item from Wishlist
User Story: As a Logged-in Buyer, I want to remove items from my wishlist, so that I can manage my saved products
Priority: Medium
Related Requirements: FR-006

GIVEN the user is logged in
  AND the user has items in their wishlist
  AND the user is viewing the wishlist page

WHEN the user clicks the "Remove" button/icon for a specific item

THEN the system should:
  - Remove the item from the wishlist immediately
  - Update the wishlist display
  - Update wishlist count in header
  - Display a confirmation message (e.g., "Removed from wishlist")
  - Optionally offer an "Undo" option

WHEN the user removes all items from wishlist

THEN the system should display "Your wishlist is empty" message
  AND provide a link to browse products

WHEN the user is on a product details page for a wishlisted item
  AND clicks the wishlisted icon to unwishlist

THEN the system should remove from wishlist
  AND update the icon state to "not wishlisted"

EXPECTED RESULT:
- Items are removed from wishlist successfully
- Wishlist updates immediately
- Wishlist count updates correctly
- Confirmation messages are clear
- Can remove from both wishlist page and product page
- Empty wishlist state is handled properly
- Undo functionality works if implemented
