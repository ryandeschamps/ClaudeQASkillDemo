TEST SCRIPT: TS-112
Test Scenario: Create Promotional Email
User Story: As a Logged-in Admin, I want to create email content for promotions and new products, so that I can market to customers
Priority: Low
Related Requirements: FR-025

GIVEN the admin is logged into the admin panel

WHEN the admin navigates to "Email Management" or "Marketing" section
  AND clicks "Create New Email Campaign"
  AND enters email information:
    - Campaign name
    - Email subject line
    - Email content (using email editor)
    - Target audience (all customers, specific segment)
    - Send date/time (immediate or scheduled)
    - Product links/images
  AND saves or schedules the email

THEN the system should:
  - Validate required fields
  - Save the email campaign
  - Display success confirmation message
  - Schedule for sending if future date selected
  - Send immediately if "Send Now" selected

WHEN creating email content

THEN the system should provide:
  - Email template editor (drag & drop or HTML)
  - Image upload capability
  - Product insertion tools
  - Personalization tokens (customer name, etc.)
  - Preview in different email clients
  - Test email sending capability

WHEN the admin previews the email

THEN the preview should show:
  - How email will look in inbox
  - Mobile and desktop views
  - All images and formatting

WHEN the admin sends a test email

THEN the system should:
  - Send to specified test email address
  - Allow verification before sending to all customers

WHEN the email is sent to customers

THEN the system should:
  - Send to selected audience
  - Track email delivery
  - Track open rates (optional)
  - Track click-through rates (optional)
  - Log sending completion

EXPECTED RESULT:
- Promotional emails can be created successfully
- Email editor provides necessary tools
- Templates make creation easier
- Preview functionality works correctly
- Test emails can be sent before campaign
- Scheduling works for future sends
- Emails are delivered successfully
- Basic analytics are available (if implemented)
