TEST SCRIPT: TS-009
Test Scenario: Sort Product Listings
User Story: As a Visitor, I want to sort products by different criteria (price, rating, newest), so that I can find products that best match my preferences
Priority: High
Related Requirements: FR-004

GIVEN the user is viewing a product listing page
  AND multiple products are available
  AND sorting options are displayed

WHEN the user selects "Price: Low to High" from the sort dropdown

THEN the product listing should reorder with lowest priced items first
  AND the sort selection should be visually indicated

WHEN the user selects "Price: High to Low"

THEN the product listing should reorder with highest priced items first

WHEN the user selects "Rating: High to Low"

THEN the product listing should reorder with highest rated items first

WHEN the user selects "Newest First"

THEN the product listing should reorder with most recently added products first

EXPECTED RESULT:
- All sort options function correctly
- Products are ordered accurately based on selected criteria
- Sort selection persists during browsing session
- Sorting works in combination with filters
- Page performance remains acceptable during sorting
- Visual indication of current sort order is clear
