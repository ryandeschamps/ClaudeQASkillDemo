TEST SCRIPT: TS-007
Test Scenario: Share Product on Social Media
User Story: As a Visitor, I want to share interesting products on social media platforms, so that I can show them to friends or save them for later reference
Priority: Low
Related Requirements: FR-009

GIVEN the user is viewing a product details page
  AND social media sharing options are configured

WHEN the user clicks on a social media share button (Facebook, Twitter, Pinterest, etc.)

THEN the system should open the appropriate social media sharing dialog
  AND pre-populate the share content with:
    - Product name
    - Product image
    - Product link
    - Brief description (if applicable)
  AND allow the user to customize the share message

WHEN the user completes the sharing process

THEN the product should be successfully shared on the selected platform
  AND the user should remain on the product details page

EXPECTED RESULT:
- Social media share buttons are visible and functional
- Share dialogs open correctly for each platform
- Product information is pre-populated accurately
- Sharing completes successfully
- User experience is seamless
- Product links shared are trackable (if analytics enabled)
