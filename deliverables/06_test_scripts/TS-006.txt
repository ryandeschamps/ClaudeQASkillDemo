TEST SCRIPT: TS-006
Test Scenario: View Product Variations
User Story: As a Visitor, I want to see available sizes and colors for a product, so that I can determine if the product meets my needs
Priority: High
Related Requirements: FR-005

GIVEN the user is viewing a product details page
  AND the product has multiple variations (sizes and/or colors)

WHEN the page loads

THEN the system should display all available size options
  AND display all available color options
  AND indicate which variations are currently in stock
  AND indicate which variations are out of stock

WHEN the user selects a specific size

THEN available colors for that size should be highlighted or enabled
  AND unavailable combinations should be disabled or grayed out

WHEN the user selects a specific color

THEN available sizes for that color should be highlighted or enabled
  AND the product image should update to show the selected color (if applicable)

EXPECTED RESULT:
- All product variations are displayed clearly
- Stock availability is accurately indicated
- Selection of size/color updates available options dynamically
- Out of stock items are clearly marked
- User interface is intuitive and responsive
- Product images update based on color selection
