TEST SCRIPT: TS-017
Test Scenario: Logout from Account
User Story: As a Logged-in Buyer, I want to log out of my account, so that I can secure my account when using shared devices
Priority: Medium
Related Requirements: FR-010

GIVEN the user is logged into their account
  AND the user is on any page of the website

WHEN the user clicks the "Logout" button or link

THEN the system should:
  - Terminate the user's session
  - Clear authentication tokens/cookies
  - Redirect to the homepage or login page
  - Display a confirmation message (optional)

WHEN the user tries to access account-specific pages after logout

THEN the system should redirect to the login page
  AND prompt the user to log in

WHEN the user clicks browser back button after logout

THEN the system should not allow access to protected pages
  AND should redirect to login page

EXPECTED RESULT:
- User is successfully logged out
- Session is completely terminated
- Authentication tokens are cleared
- User is redirected appropriately
- Cannot access protected pages without re-authentication
- Back button does not expose authenticated content
- Security: session is properly invalidated on server side
