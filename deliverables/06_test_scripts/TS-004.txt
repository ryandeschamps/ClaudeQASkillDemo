TEST SCRIPT: TS-004
Test Scenario: Check Shipping Availability by PIN Code
User Story: As a Visitor, I want to check if a product can be shipped to my PIN code, so that I can know if I can order the product before registering
Priority: High
Related Requirements: FR-005

GIVEN the user is viewing a product details page
  AND the user is not logged in
  AND the product has shipping availability data configured

WHEN the user enters a valid PIN code in the availability checker
  AND clicks the "Check" button

THEN the system should validate the PIN code
  AND display availability status (Available/Not Available)
  AND show estimated delivery date if available
  AND display shipping charges if applicable

WHEN the user enters an invalid PIN code format

THEN the system should display an appropriate error message
  AND prompt the user to enter a valid PIN code

EXPECTED RESULT:
- PIN code validation works correctly
- Availability status is accurate based on configured data
- Estimated delivery dates are shown when available
- Shipping charges are displayed correctly
- Error handling for invalid PIN codes is functional
- Response time is within acceptable limits
