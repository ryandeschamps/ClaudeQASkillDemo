TEST SCRIPT: TS-023
Test Scenario: View Wishlist
User Story: As a Logged-in Buyer, I want to view all products in my wishlist, so that I can review items I saved for later
Priority: Medium
Related Requirements: FR-006

GIVEN the user is logged in
  AND the user has items in their wishlist

WHEN the user clicks on the wishlist icon or navigates to the wishlist page

THEN the system should display the wishlist page with:
  - List of all wishlisted products showing:
    * Product image
    * Product name
    * Current price
    * Product rating
    * Availability status
  - Option to remove items from wishlist
  - Option to move items to cart
  - Option to view product details

WHEN the wishlist is empty

THEN the system should display "Your wishlist is empty" message
  AND provide a link to browse products

WHEN a wishlisted product is out of stock

THEN the system should clearly indicate "Out of Stock" status
  AND optionally offer notification when back in stock

EXPECTED RESULT:
- All wishlisted items are displayed correctly
- Product information is current and accurate
- Price updates are reflected in wishlist
- Out of stock items are clearly marked
- Empty wishlist is handled gracefully
- Navigation to product details works
- Page loads within acceptable time
