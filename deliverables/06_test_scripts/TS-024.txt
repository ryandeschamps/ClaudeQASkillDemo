TEST SCRIPT: TS-024
Test Scenario: Move Wishlist Item to Cart
User Story: As a Logged-in Buyer, I want to move items from wishlist to shopping cart, so that I can purchase saved items
Priority: Medium
Related Requirements: FR-006

GIVEN the user is logged in
  AND the user has items in their wishlist
  AND the user is viewing the wishlist page

WHEN the user clicks "Add to Cart" for a wishlisted product
  AND selects size and color (if required)

THEN the system should:
  - Add the product with selected attributes to the shopping cart
  - Display a confirmation message
  - Optionally remove the item from wishlist
  - Update cart count in header

WHEN the product requires size/color selection

THEN the system should prompt the user to select these attributes
  AND not add to cart until selections are made

WHEN the product is out of stock

THEN the system should display an error message
  AND prevent adding to cart

EXPECTED RESULT:
- Products move from wishlist to cart successfully
- Required attributes (size/color) are captured
- Cart count updates correctly
- Confirmation messages are clear
- Out of stock items cannot be added to cart
- User can continue shopping or proceed to checkout
- Wishlist updates appropriately after moving items
