Test Script: TS-002 - Email Verification After Registration

GIVEN a user has registered but not verified their email
AND user received email verification link
WHEN user clicks on the email verification link
THEN the system should validate the verification token
AND activate the user account
AND update email verification status to "Verified"
AND display success message "Email verified successfully! You can now log in"
AND redirect to login page

EXPECTED RESULT:
- User account status changed from "Unverified" to "Verified"
- User can now log in with credentials
- Verification token marked as used
