Test Script: TS-003 - Registration with Invalid Email Format

GIVEN a visitor is on the registration page
WHEN the visitor enters an invalid email format (e.g., "invalid-email")
AND fills other required fields correctly
AND clicks "Register" button
THEN the system should display validation error "Please enter a valid email address"
AND prevent form submission
AND keep the user on the registration page

EXPECTED RESULT:
- No account created
- Error message displayed inline next to email field
- All other field values retained
