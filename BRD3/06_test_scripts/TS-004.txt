Test Script: TS-004 - Registration with Mismatched Passwords

GIVEN a visitor is on the registration page
WHEN the visitor enters Password: "SecurePass123!"
AND enters Confirm Password: "DifferentPass456!"
AND fills other required fields correctly
AND clicks "Register" button
THEN the system should display validation error "Passwords do not match"
AND prevent form submission

EXPECTED RESULT:
- No account created
- Error message displayed below password fields
- Password fields cleared for security
