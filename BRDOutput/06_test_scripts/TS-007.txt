Test Script: TS-007 - Search Products by Keyword

Test Scenario: As a visitor or buyer, I want to search for products using keywords, so that I can quickly find specific items I'm looking for.

Prerequisites:
- Website is accessible
- Product catalog has products available
- Search functionality is operational

Test Data:
- Search Keyword: "blue jeans"
- Expected Results: Products containing "blue" and/or "jeans"

Test Steps:

GIVEN the user is on the website homepage
WHEN the user views the page
THEN the page should display a search bar in the header
AND the search bar should have a text input field
AND the search bar should have a search button or icon

GIVEN the user sees the search bar
WHEN the user clicks on the search input field
THEN the field should become focused and ready for input

GIVEN the search input field is focused
WHEN the user types "blue jeans" into the search field
AND the user clicks the search button (or presses Enter)
THEN the system should process the search query
AND the system should search for products matching "blue" and "jeans" keywords
AND the user should be redirected to the search results page

GIVEN the user is on the search results page
WHEN the page loads
THEN the page should display:
  - Search query "blue jeans" indicated (e.g., "Search results for: blue jeans")
  - Number of products found (e.g., "15 results found")
  - List of matching products
AND each product in the list should display:
  - Product thumbnail image
  - Product title
  - Price
  - Ratings and reviews

GIVEN the search results are displayed
WHEN the user reviews the results
THEN all displayed products should be relevant to "blue jeans"
AND products should include items like:
  - Blue Denim Jeans
  - Straight Cut Blue Jeans
  - Slim Fit Blue Jeans
  - (any jeans in blue color or with "blue" in name/description)

GIVEN the user sees the search results
WHEN the user clicks on a product title or image
THEN the user should be navigated to that product's detail page

Expected Result:
- Search bar is visible and accessible from homepage
- User can enter search keywords
- System processes search query when user submits
- Search results page displays with query and result count
- All matching products are displayed with key information
- Products are relevant to the search keywords
- User can click on products to view details
- Search works for both logged-in and guest users

Validation Points:
- Search bar accepts text input
- Search can be triggered by button click or Enter key
- Search query is processed correctly
- Results page displays search query
- Result count is accurate
- Products match search keywords in title, description, or attributes
- Product listings show thumbnail, title, price, ratings
- Clicking on products navigates to detail pages
- Search works without requiring login
- Empty search shows appropriate message
- No results shows "No products found" message
