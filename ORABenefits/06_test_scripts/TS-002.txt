Test Script ID: TS-002
Test Scenario: New Hire with Default Enrollment
Priority: Critical
Preconditions:
- Default benefit plan rules configured in system
- New hire enrollment period has expired
- Employee did not make active elections during enrollment window
- Default enrollment batch process scheduled and functional

GIVEN: A new employee (Howard Hughes) hired on 2024-05-15 did not complete benefits enrollment within the 31-day window
WHEN: The default enrollment batch process runs after the enrollment deadline
THEN: The employee is automatically enrolled in default benefit plans ensuring basic coverage

Test Steps:
1. Create new hire record for employee with hire date 2024-05-15
2. Verify employee receives new hire enrollment notification
3. Do not complete any benefits enrollment actions as the employee
4. Wait for enrollment period to expire (31 days after hire date)
5. As Benefits Administrator, navigate to "Default Enrollment Processing"
6. Run "Process Default Enrollment" batch job for the specified benefits group (BG-001)
7. Verify batch job completes successfully without errors
8. Review batch job results and confirm employee processed for default enrollment
9. Navigate to employee's benefits summary page
10. Verify default plan enrollment: Platinum HMO Family coverage
11. Verify monthly premium calculated correctly ($610.00)
12. Verify coverage effective date set appropriately
13. Check that employee receives notification of default enrollment
14. Verify payroll deduction setup for premium collection
15. Confirm employee can view their default enrollment in self-service portal

Expected Result:
- Employee automatically enrolled in Platinum HMO with Family coverage
- Default enrollment status clearly indicated in system
- Monthly premium of $610.00 calculated and assigned
- Employee notified of default enrollment via email
- Coverage effective date established per default rules
- Employee retains ability to make changes during next enrollment period
- No action items or certifications required for default enrollment
