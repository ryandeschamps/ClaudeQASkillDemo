Test Script ID: TS-048
Test Scenario: Administrator Extend Document Validity
Priority: Medium
Preconditions:
- Benefits administrator role with document management authority
- Previously approved document approaching or past expiration
- Document validity extension capability enabled
- Valid business reason for extension
- Audit logging for document validity changes

GIVEN: An employee's approved marriage certificate is approaching expiration but circumstances warrant extension
WHEN: Benefits administrator extends the document validity period
THEN: Document validity period successfully extended and employee not required to resubmit

Test Steps:
1. Employee's marriage certificate approved 12 months ago (2023-11-01)
2. Standard validity period: 12 months
3. Document set to expire: 2024-11-01
4. Special circumstance: Employee on extended international assignment, difficult to obtain new certified copy
5. Benefits administrator logs into Oracle Fusion HCM
6. Navigate to employee's benefits record
7. Select "Documents" or "Certifications" section
8. Review list of employee's uploaded documents:
   - Marriage certificate: Approved 2023-11-01, Expires 2024-11-01
9. Click on marriage certificate to view details
10. Click "Extend Validity Period" button
11. System displays current expiration date: 2024-11-01
12. System prompts for new expiration date
13. Enter new expiration date: 2025-11-01 (extending by 12 months)
14. System prompts for extension reason
15. Select reason code: "Employee on International Assignment"
16. Enter detailed notes: "Employee on 2-year international assignment, certified documents difficult to obtain. Management approved extension."
17. Upload approval documentation if required (email approval from HR Director)
18. Review extension summary:
    - Original expiration: 2024-11-01
    - New expiration: 2025-11-01
    - Extension period: 12 months
    - Reason: International assignment
19. Confirm validity extension
20. System updates document expiration date
21. Document now valid until 2025-11-01
22. Employee receives notification: "Your marriage certificate validity has been extended"
23. No action required by employee
24. Audit trail captures: extension date, new expiration, reason, approver, authorization

Expected Result:
- Administrator successfully accesses document management functions
- Current document validity and expiration dates clearly displayed
- Extension capability available with proper authorization
- Administrator can specify new expiration date
- Extension reason required and captured
- Approval documentation can be attached
- Extension summary clear before confirmation
- Document expiration date successfully updated
- Employee notified of extension (no action required)
- Document continues to be valid for enrollments and reuse
- Employee not burdened with resubmission during extension period
- Audit trail comprehensively documents extension with justification
- Extension is exception-based with proper oversight

Test Data Reference:
- Any document approaching or past expiration
- Requires valid business reason for extension
- Administrator with document management privileges required

Notes:
- Verify document validity extension requires proper authorization
- Confirm extension is exception-based with documented justification
- Test various extension reasons: international assignment, hardship, etc.
- Verify extension period reasonable (not indefinite extensions)
- Ensure employee communication about extension
- Test that extended documents function same as originally valid documents
- Confirm periodic review of extensions for compliance
- Verify extensions tracked separately from original approvals
- Test management oversight of extension activity
- Ensure extensions don't circumvent necessary verification processes
