Test Script ID: TS-043
Test Scenario: Administrator Run Diagnostic Reports
Priority: Medium
Preconditions:
- Benefits administrator role with diagnostic reporting access
- Diagnostic tools configured and functional
- Employee experiencing enrollment or eligibility issues
- Comprehensive diagnostic checks available
- Report output provides actionable information

GIVEN: An employee reports benefits enrollment issues and root cause is unclear
WHEN: Benefits administrator runs diagnostic health check for the participant
THEN: Diagnostic report identifies data problems, configuration issues, or processing errors requiring resolution

Test Steps:
1. Employee reports issue: "I can't see any benefit plans when trying to enroll"
2. Benefits administrator logs into Oracle Fusion HCM
3. Navigate to "Benefits Administration" work area
4. Select "Diagnostics" or "Health Check" tools
5. Enter employee identifier: Name or Person Number
6. Click "Run Benefits Health Check" or "Run Diagnostics"
7. System performs comprehensive diagnostic checks:
   - Employment status: Active, Terminated, On Leave?
   - Benefits eligibility: Eligible, Ineligible, Pending?
   - Eligibility rules: Which rules apply? Are they met?
   - Benefits group assignment: Assigned? Correct group?
   - Benefits relationship: Properly assigned?
   - Life events: Any active or pending life events?
   - Enrollment periods: Any active enrollment windows?
   - Plan configurations: Plans active and available?
   - Coverage configurations: Coverage levels configured?
   - Rate tables: Rates defined for plan/coverage combinations?
8. Diagnostic report generated showing results for each check
9. Review diagnostic findings:
   - Employment status: Active ✓
   - Benefits eligibility: Eligible ✓
   - Eligibility rules: Met ✓
   - Benefits group: BG-001 assigned ✓
   - Benefits relationship: Assigned ✓
   - Life events: None active ✓
   - Enrollment periods: ISSUE FOUND - No active enrollment period ✗
   - Plan configurations: Active ✓
   - Rate tables: ISSUE FOUND - Missing rates for Bronze PPO Family ✗
10. Issues identified:
    - Employee outside enrollment period (no life event)
    - Missing rate table entry for one plan/coverage combination
11. Review recommended actions:
    - Check if life event should be reported
    - Verify rate table completeness
12. Administrator investigates further based on diagnostic findings
13. Resolve identified issues (report life event, add missing rates)
14. Re-run diagnostic to confirm resolution
15. All checks now pass ✓
16. Employee can now access benefit plans and enroll

Expected Result:
- Diagnostic tool easily accessible to benefits administrators
- Comprehensive health check covers all key enrollment factors
- Diagnostic runs quickly and produces clear report
- Each diagnostic check clearly labeled with pass/fail status
- Issues highlighted prominently in report
- Recommended actions provided for issue resolution
- Report identifies root cause of enrollment problems
- Administrator can use findings to resolve issues systematically
- Diagnostic can be re-run to verify resolution
- Report exportable or printable for documentation
- Common issues (missing rates, no enrollment period, eligibility failures) detected
- Diagnostic tool accelerates troubleshooting significantly

Test Data Reference:
- Any employee experiencing enrollment or eligibility issues
- Various issue types: eligibility problems, missing configuration, data errors
- Diagnostic tool should cover all common problem areas

Notes:
- Verify diagnostic tool comprehensive in checking all enrollment factors
- Confirm report output is clear and actionable for administrators
- Test various issue scenarios: eligibility failures, missing rates, inactive plans, etc.
- Verify recommended actions are helpful and accurate
- Ensure diagnostic tool accessible from multiple admin work areas
- Test that diagnostic can run on any employee
- Confirm diagnostic identifies both data and configuration issues
- Verify diagnostic does not modify data, only reports findings
- Test that complex issues (multiple problems) properly identified
- Ensure diagnostic tool enhances troubleshooting efficiency significantly
