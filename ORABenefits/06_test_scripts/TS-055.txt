Test Script ID: TS-055
Test Scenario: Contingent Beneficiary Designation
Priority: Medium
Preconditions:
- Employee enrolled in life insurance
- Primary beneficiaries already designated
- Contingent beneficiary functionality enabled
- Understanding of contingent vs. primary beneficiary roles
- Percentage validation for contingent beneficiaries

GIVEN: An employee has designated primary beneficiaries and wants to add contingent beneficiaries as backup
WHEN: The employee designates contingent beneficiaries who will receive proceeds if primary beneficiaries predecease
THEN: Contingent beneficiaries successfully designated with proper understanding of their role

Test Steps:
1. Employee logs into self-service portal
2. Navigate to "Beneficiaries" section
3. Review existing primary beneficiaries:
   - Primary: Emma Martinez (Spouse) - 100%
4. Click "Add Contingent Beneficiaries" or "Add Backup Beneficiaries"
5. System explains contingent beneficiary role:
   - "Contingent beneficiaries receive proceeds only if all primary beneficiaries predecease you"
   - "This provides a backup designation"
6. Click "Add Contingent Beneficiary"
7. Add first contingent beneficiary:
   - Name: Robert Martinez (Son)
   - Relationship: Child
   - Date of Birth: 1990-05-10
   - SSN: XXX-XX-XXXX
   - Percentage: 50%
8. Click "Add Another Contingent Beneficiary"
9. Add second contingent beneficiary:
   - Name: Maria Martinez (Daughter)
   - Relationship: Child
   - Date of Birth: 1992-08-15
   - SSN: XXX-XX-XXXX
   - Percentage: 50%
10. System validates contingent beneficiary percentages: 50% + 50% = 100% âœ“
11. Review complete beneficiary designation:
    - PRIMARY BENEFICIARIES:
      - Emma Martinez (Spouse) - 100%
    - CONTINGENT BENEFICIARIES (if primary predeceases):
      - Robert Martinez (Son) - 50%
      - Maria Martinez (Daughter) - 50%
12. System displays benefit flow explanation:
    - "If Emma Martinez survives you, she receives 100% of proceeds"
    - "If Emma Martinez predeceases you, Robert and Maria each receive 50%"
13. Review and confirm understanding of contingent beneficiary role
14. Electronic signature for beneficiary designation
15. Type full name to sign
16. Submit designation with both primary and contingent beneficiaries
17. Confirmation: "Primary and contingent beneficiaries designated successfully"
18. Receive confirmation email with complete beneficiary structure

Expected Result:
- Employee can add contingent beneficiaries after primary beneficiaries
- System explains contingent beneficiary role clearly
- Contingent beneficiary form same as primary beneficiary form
- Multiple contingent beneficiaries can be added
- Contingent beneficiary percentages must total 100%
- System validates contingent percentages separately from primary
- Complete beneficiary summary shows hierarchy: primary vs. contingent
- Benefit flow explanation clarifies when contingent beneficiaries receive proceeds
- Employee understands contingent beneficiaries are backup only
- All beneficiaries saved successfully with proper designation type
- Confirmation clearly distinguishes primary from contingent beneficiaries
- Beneficiary designation provides comprehensive backup plan

Test Data Reference:
- Variant IDs: Life insurance scenarios with contingent beneficiaries
- Common structures:
  - Primary: Spouse 100%, Contingent: Children split equally
  - Primary: Multiple children split, Contingent: Siblings or parents
  - Primary: Trust, Contingent: Individual family members

Notes:
- Verify clear explanation of contingent beneficiary role
- Confirm contingent percentages validated separately from primary
- Test that contingent beneficiaries only receive if ALL primary beneficiaries predecease
- Verify system explains benefit flow clearly
- Ensure employees understand contingent is backup, not concurrent
- Test scenarios with no contingent beneficiaries (optional)
- Confirm contingent beneficiaries can be updated independently of primary
- Verify estate is default contingent if none designated
- Test that minors can be contingent beneficiaries
- Ensure per stirpes option available for contingent child beneficiaries
