Test Script ID: TS-016
Test Scenario: Address Change
Priority: High
Preconditions:
- Active employee enrolled in HMO plan with location-specific network
- Address change triggers life event in system
- Plan options vary by geographic location
- New address is outside current HMO service area
- PPO or other plan options available in new location

GIVEN: An employee (Michelle Walker) relocates from one state to another on 2024-10-15
WHEN: The employee updates their address and the system triggers a life event for plan change due to location
THEN: Employee can change HMO plan to one available in new location or switch to location-independent plan

Test Steps:
1. Log into self-service portal as employee (michelle.walker@company.com)
2. Navigate to "Personal Information" or "My Contact Information"
3. Click "Edit" or "Update Address"
4. Enter new address in different state with different HMO service area
5. Enter effective date of address change: 2024-10-15
6. Save address change
7. System validates address and identifies HMO network issue
8. System automatically triggers "Address Change" life event
9. Notification displays: "Your HMO plan is not available in your new location"
10. System offers options: change to available HMO in new area or switch to PPO
11. Click "Review Plan Options" or "Make Plan Changes"
12. Review current plan: Silver HMO Employee Only ($180.00)
13. View available plans in new location
14. Select new plan option (e.g., Gold PPO or Silver HMO in new service area)
15. Review new premium if different from current
16. Submit plan change election
17. Verify coverage effective date aligned with address change (2024-10-15)
18. Receive confirmation of plan change and new provider network information
19. Update provider directory access for new location
20. Verify new plan ID card issued for new coverage

Expected Result:
- Address change successfully updated in system
- System identifies HMO network availability issue
- Address change life event automatically triggered
- Employee notified of need to change health plan
- Plan options in new location clearly displayed
- Employee successfully changes to available plan
- Coverage effective date matches address change date (2024-10-15)
- New plan premium communicated and payroll updated
- New provider network information provided
- New insurance ID card issued
- No coverage gap during transition
